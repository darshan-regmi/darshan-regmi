### <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Travel%20and%20places/Glowing%20Star.png" alt="star" width="25" height="25" /> Veon — Independent Web App Store

[![GitHub](https://img.shields.io/badge/GitHub-veon-181717?style=flat-square&logo=github)](https://github.com/darshan-regmi/veon) [![Live](https://img.shields.io/badge/Live-veon.pages.dev-00C7B7?style=flat-square&logo=cloudflare)](https://veon.pages.dev)

Web-based platform to distribute Android APKs with custom branding, version tracking, and an admin panel.

- Built with **Next.js 15**, **Firebase**, and **Cloudflare Pages**
- Auto-fetches GitHub releases for automated version management
- CI/CD deployment pipeline; unlimited APK storage via Firebase
